## üöÄ Quick Start with v1.1

### Installation

```bash
# Frontend - install new dependencies
cd frontend
npm install  # Includes monaco-editor, diff

# Backend - no new dependencies needed
cd backend
# Already has all required packages
```

### Using New Features

**1. Code Viewing:**
```typescript
import CodeViewer from '@/components/CodeViewer';

<CodeViewer 
  code={sourceCode}
  language="python"
  filePath="example.py"
  highlightLine={42}
/>
```

**2. Diff Viewing:**
```typescript
import DiffViewer from '@/components/DiffViewer';

<DiffViewer
  original={originalCode}
  modified={modifiedCode}
  fileName="patch.py"
  viewMode="unified"
/>
```

**3. WebSocket Streaming:**
```typescript
const ws = new WebSocket('ws://localhost:8000/ws/chat/{sessionId}');
ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  // Handle real-time updates
};
```

**4. Symbol Lineage:**
```bash
curl http://localhost:8000/symbols/lineage/UserAuth
```

**5. CFG Analysis:**
```bash
curl http://localhost:8000/cfg/function/process_payment?file_path=payment.py
```
